# GitScope

GitScope is a full-stack web application that analyzes GitHub profiles and repositories using the GitHub API. Users can search for GitHub accounts and view repository information, developer activity, and profile insights through an interactive dashboard.

# Tech Stack

- Frontend: React.js, Vite, Tailwind CSS
- Backend: Node.js, Express.js
- Database: MySQL
- API: GitHub REST API

# Features

- GitHub profile search
- Repository analysis
- Real-time GitHub data
- Responsive UI
- Database integration

# Setup

1. Install dependencies:

npm install

2. Configure ".env":

- DB_HOST=localhost
- DB_USER=root
- DB_PASSWORD=your_password
- DB_NAME=gitscope
- GITHUB_TOKEN=your_token

3. Start the application:

npm run dev

| Stars | 25% |
| Forks | 10% |
| Account Age | 15% |

**Levels:** Beginner (0–24) · Intermediate (25–49) · Advanced (50–74) · Expert (75–100)
